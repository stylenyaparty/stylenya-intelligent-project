# Getting Started

This documentation site is generated with Mintlify and lives in the repository under the docs folder.

## Local preview

From the repository root:

```bash
npx mintlify dev
```

If the CLI is installed globally, you can also run:

```bash
mintlify dev
```

## Structure

Key documentation sources:

- Architecture and domain: docs/system-architecture.md and docs/domain-model.md
- Data model: docs/data-model.md
- API contract: docs/api-contract.md
- App-specific notes: apps/api/README.md and apps/web/README.md
