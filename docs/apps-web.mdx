# Web App

The Web app provides the decision-support dashboard. It consumes the API and focuses on UX and data visualization.

## Key areas

- App entry: apps/web/src/main.tsx
- Pages and routes: apps/web/src/pages
- Shared components: apps/web/src/components
- API client layer: apps/web/src/api

## Design goals

- Clear, actionable insights
- Fast KPI exploration
- Traceable recommendations
